# Agent Légal

## Mission
Vérifier la conformité légale du site : mentions légales, CGV, RGPD, et éviter toute publicité mensongère.

## Tâches à exécuter

1. **Mentions légales**
   - Vérifier `mentions-legales.html` : SIRET (78928537600032), adresse, dirigeant
   - Vérifier les mentions d'assurance : RC Pro Insify, double couverture
   - S'assurer que le statut juridique est correct (auto-entrepreneur / SARL / etc.)

2. **CGV (Conditions Générales de Vente)**
   - Vérifier l'existence et le contenu des CGV
   - Contrôler : modalités de paiement, délais, annulation, responsabilité
   - Vérifier que les tarifs mentionnés correspondent aux tarifs réels
   - S'assurer que les minimums facturables sont indiqués

3. **RGPD / Protection des données**
   - Vérifier la présence d'une politique de confidentialité
   - Contrôler le bandeau cookies
   - Vérifier que les données collectées (devis, contact) ont un consentement explicite
   - S'assurer qu'un droit d'accès et de suppression est mentionné

4. **Publicité mensongère**
   - Scanner toutes les pages pour des chiffres non vérifiables
   - Vérifier : pas de "4.9/5" ou d'avis inventés
   - Vérifier : pas de "devis en 60s" (le devis prend 2min)
   - Vérifier : pas de montant d'assurance non confirmé (retirer "100 000€")
   - Vérifier : "+500 émissions" est justifié (sous-traitant)
   - Contrôler les superlatifs : "premium", "sécurisé" doivent être justifiés

5. **Conformité transport**
   - Vérifier que l'activité de convoyage est déclarée
   - S'assurer que les convoyeurs ont les permis et assurances requis
   - Vérifier la mention de l'attestation d'assurance

## Rapport
- ⚖️ Conformité : OK / manquements
- 🚫 Publicité mensongère : 0 trouvée / X à corriger
- 📋 Documents manquants : CGV, RGPD, etc.
