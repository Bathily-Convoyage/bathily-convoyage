# Agent SEO

## Mission
Optimiser le référencement naturel de Bathily-Convoyage sur Google.

## Tâches à exécuter

1. **Balises meta**
   - Vérifier `<title>` sur toutes les pages (unique, < 60 caractères)
   - Vérifier `<meta description>` (unique, < 160 caractères, avec mots-clés)
   - Vérifier les balises Open Graph (`og:title`, `og:description`, `og:image`)
   - Vérifier les balises Twitter Card

2. **Pages villes (39 fichiers)**
   - Vérifier que chaque page cible une ville + mots-clés (convoyage, transport voiture)
   - S'assurer que le H1 contient la ville
   - Vérifier la structure H1 → H2 → H3
   - Contrôler le contenu unique par page (pas de duplicate content)

3. **Sitemap et robots.txt**
   - Vérifier l'existence de `sitemap.xml`
   - Vérifier que toutes les pages sont incluses
   - Vérifier `robots.txt` : autoriser le crawl, bloquer les pages admin

4. **Liens internes**
   - Vérifier que toutes les pages ont un lien vers `devis.html`
   - Vérifier la présence de liens croisés entre pages villes
   - Contrôler les liens footer (mentions légales, contact, etc.)

5. **Performance SEO**
   - Vérifier le temps de chargement (images optimisées, CSS minifié)
   - Vérifier la version mobile (responsive)
   - Contrôler les images : `alt` text, format WebP si possible

6. **Données structurées**
   - Vérifier la présence de schema.org JSON-LD (LocalBusiness, Service)
   - S'assurer que le SIRET, adresse et téléphone sont structurés

7. **Mots-clés cibles**
   - "convoyage voiture" + ville
   - "transport véhicule" + ville
   - "convoyage auto moto France"
   - "convoyeur professionnel"

## Rapport
- 📈 Score SEO par page
- 🏷️ Balises manquantes
- 🔗 Liens internes cassés
- 🗺️ Sitemap : OK / à mettre à jour
