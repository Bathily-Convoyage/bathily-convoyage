## POST 1 - Instagram - 29 juin 2026 à 9h

### Étape 1 : Créer le post
1. Connecte-toi à Buffer
2. Clique sur **"Publishing"** → **"Create Post"**
3. Sélectionne **Instagram** dans les plateformes

### Étape 2 : Upload le visuel
1. Clique sur **"Upload media"**
2. Navigue vers : `social-media/assets/`
3. Sélectionne : `Plaquette-v2.png`
4. Clique sur **"Open"**

### Étape 3 : Colle la légence
```
Découvrez notre nouvelle plaquette commerciale ! Convoyage auto & moto partout en France avec suivi GPS et accompagnement premium. #convoyage #transportvehicule #france
```

### Étape 4 : Configure la date
1. Clique sur **"Schedule"**
2. Date : **29 juin 2026**
3. Heure : **9h00**
4. Clique sur **"Confirm"**

### Étape 5 : Ajoute à la queue
1. Clique sur **"Add to Queue"**
2. Le post est maintenant planifié !

## Problèmes fréquents

### Le visuel ne s’upload pas
- Vérifie que le fichier est bien dans `social-media/assets/`
- Le fichier doit être en PNG, JPG ou WEBP

### La légence est trop longue
- Réduis le nombre de hashtags
- Buffer a une limite de caractères par plateforme

### Le post ne se publie pas
- Vérifie que le compte est connecté à Buffer
- Vérifie les permissions du compte
